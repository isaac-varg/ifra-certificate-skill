# IFRA Certificate of Conformity Skill

A AI Agent skill that generates a branded IFRA Certificates of Conformity as a PDF. Supports single fragrance/essential oil IFRAs or blends of oils with mathematically determined usage limits based on the individual components.

## Features

- **Single Oil** 
  - parse a supplier's IFRA PDF and reissue it under your company's branding
- **Multi-Oil Blend**
  - combine multiple components into one finished certificate with blended usage limits
- Branded header with company logo, name, phone, and email
- Full IFRA 51th Amendment category definitions
- Disclaimer page
- Per-page footer

## Setup

Rename the example config and fill in your details:

```bash
mv config.example.json config.json
```

```json
{
  "company_name": "Your Company Name, Inc.",
  "phone": "(555) 123-4567",
  "email": "hello@yourcompany.com",
  "logo": "assets/logo.png",
  "amendment": "51th"
}
```

Replace `assets/logo.png` with your company logo (update the `logo` path in config if the filename differs).
